## 화면별 권장 조합
| 화면 | 사용 ID |
|---|---|
| 홈 (SCR-C-01) | `market_id=f1a2b3c4-...` |
| 상품 상세 (SCR-C-03) | `product_id=p1000001-...` |
| 점포 상세 + 스토리 | `store_id=d1000001-...` (스토리 게시됨) |
| 드랍 리스트 (SCR-C-04) | `market_id=f1a2b3c4-...&status=arrived` |
| 상인 대시보드 (SCR-M-01) | `merchant01@market.com` 로 로그인 후 `/merchant/dashboard` |
| 상인 스토리 화면 (SCR-M-04) | `merchant01@market.com` + `store_id=d1000001-...` |
| 가격 정책 보조 (대시보드) | `merchant01@market.com` + `/merchant/dashboard/price-suggestions` |
| 가격 이력 그래프 | `merchant01@market.com` + `product_id=p1000001-...` |
